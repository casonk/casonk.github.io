`nordility` is a focused Python project extracted from `citegres` that automates NordVPN actions through a clean CLI.

It supports connecting, disconnecting, and rotating to a new country or server group, with both Windows (`NordVPN.exe`) and Linux (`nordvpn`) backends selectable at runtime. No third-party runtime dependencies are required.
