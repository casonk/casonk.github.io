`archility` is the shared architecture tooling repo across the broader portfolio. It centralizes bootstrap and rendering for PlantUML and Draw.io, provides deterministic starter architecture scaffolding, and supports richer agent-authored repository architecture.

It is used to keep repository diagrams and contributor architecture documents aligned with actual code and content structure.
