`dyno-lab` is a shared Python library that provides reusable testing infrastructure drawn from patterns across every tested repository in the portfolio.

Modules cover subprocess mocking, environment patching, filesystem fixtures, CLI output capture, HTTP session mocking, schema/contract parity checks, dynamic module loading, shared pytest markers, and smoke-test scaffolding.

It is designed to be installed as a local editable dependency and used as a drop-in test harness across any portfolio project.
