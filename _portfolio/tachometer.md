`tachometer` extracts shared profiling helpers into a reusable portfolio utility. It captures host and repository snapshots, profiles commands with pre and post samples, and writes portable JSON summaries from a manifest-driven workflow.

The repo is designed to make profiling conventions consistent across codebases without pushing workload-specific logic into the profiling layer itself.
