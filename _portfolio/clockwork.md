`clockwork` is the shared scheduling repo for recurring portfolio jobs. It keeps workload logic in the owning repositories while centralizing manifest-driven cron and `systemd` definitions, rendered artifacts, and install guidance.

The repo also includes a small web interface for browsing, enabling, and disabling scheduled jobs from the same manifest layer.
