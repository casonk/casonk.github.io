`terminility` is a shell-based setup tool that installs and fully configures tmux with mouse support, Vim-style navigation, and automatic session persistence.

It uses TPM to wire in `tmux-resurrect` and `tmux-continuum`, scans a configured git repo base directory to create a named tmux session per repository, and generates shell aliases so you can jump into any session by name.
