I’m Cason Konzer, a Data Systems Engineer at Robert Bosch LLC with graduate work in Computer Science & Information Systems at the University of Michigan-Flint.

My public work spans data systems, simulation, developer tooling, and infrastructure automation. Recent repositories include social network simulation software, portfolio-wide testing and architecture tooling, schedulers and profiling utilities, and private-network or messaging infrastructure.

## Current repo suite

- [archility](/portfolio/archility/), [dyno-lab](/portfolio/dyno-lab/), [tachometer](/portfolio/tachometer/), and [clockwork](/portfolio/clockwork/) cover architecture tooling, test infrastructure, profiling, and scheduler automation.
- [shock-relay](/portfolio/shock-relay/), [short-circuit](/portfolio/short-circuit/), [snowbridge](/portfolio/snowbridge/), and [wiring-harness](/portfolio/wiring-harness/) cover messaging, VPN access, private file sharing, and ingress for private services.
- [nordility](/portfolio/nordility/), [terminility](/portfolio/terminility/), and [crew-chief](/portfolio/crew-chief/) cover local workflow, environment setup, and local LLM service automation.

## Featured work

- [sonetsim](/publication/2024-03-28-sonetsim): a Python library for social network simulation, published on PyPI and maintained on GitHub.
- [archility](/portfolio/archility/): shared architecture tooling for repository inventory, starter scaffolding, and PlantUML/Draw.io render orchestration.
- [Bosch Trailering Safety Tool](/portfolio/bosch-trailering-safety-tool/): Bosch-sponsored capstone work using Python, PyTorch, and Docker.
- [dyno-lab](/portfolio/dyno-lab/): reusable test fixtures, mocks, and smoke-test scaffolding used across the broader portfolio.
- [wiring-harness](/portfolio/wiring-harness/): shared Caddy, mTLS, and DNS provisioning for private service ingress.
- [EcoCAR EV Challenge Data Collection Workshop](/talks/2024-04-02-ecocar-data-collection-workshop/): Bosch software team workshop support around test data setup and collection.
