`sonetsim` is a Python library for social network simulation. The package is published on PyPI and maintained in a public GitHub repository.

Links:

- [PyPI package](https://pypi.org/project/sonetsim/)
- [GitHub repository](https://github.com/casonk/sonetsim)

This project turns simulation work into a reusable software package with a public release and open source development history.
